# Icon

icon图标库
```js
import { Icon } from 'ml-ui'

Vue.component(`ml-${Icon.name}`, Icon)
```

Demo
```html
<ml-icon icon="back"></ml-icon>
```
Props
| 参数          | 说明            | 类型            | 可选值                 | 默认值   |
|-------------  |---------------- |---------------- |---------------------- |-------- |
| icon         | 名称   | String  | - | - |
